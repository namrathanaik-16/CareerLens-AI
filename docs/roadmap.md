# CareerLensAI roadmap
# Version 1(MVP)
Goal:
Build the foundation of the platform.

Features:

* User Authentication
* Resume Upload
* Resume Parsing
* Gmail Integration
* Email Synchronization
* Email Classification
* Application Tracking
* Basic Dashboard

Expected Outcome:
Users can connect their dedicated job-search email and automatically track applications.

---

## Version 2

Goal:
Help users understand how well they match a job.

Features:

* Job Description Upload
* Skill Extraction
* Resume vs Job Description Matching
* Match Score Generation
* Missing Skill Detection

Expected Outcome:
Users receive a compatibility score and skill gap analysis for each job.

---

## Version 3

Goal:
Understand rejection patterns.

Features:

* Rejection Email Analysis
* Rejection Categorization
* Missing Skill Identification
* Hiring Trend Detection

Expected Outcome:
Users understand why applications are being rejected.

---

## Version 4

Goal:
Provide career analytics.

Features:

* Application Funnel Analytics
* Response Rate Tracking
* Interview Rate Tracking
* Offer Rate Tracking
* Company Analytics
* Role-Based Analytics

Expected Outcome:
Users gain data-driven insights into their job search performance.

---

## Version 5

Goal:
Become an AI Career Strategist.

Features:

* Personalized Recommendations
* Career Roadmaps
* Skill Development Plans
* Job Search Optimization
* AI Career Advisor

Expected Outcome:
Users receive actionable strategies to improve their chances of getting hired.

---
